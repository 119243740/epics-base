EPICS Base R3.14.11: Known Problems

  • The libCom test suite program epicsCalcTest test #66 fails on some linux systems. This is not a bug in EPICS and will not be fixed; it actually demonstrates that the GCC optimizer is not generating the same result for the expression isinf(-Inf) that the glibc function returns at runtime. Both versions return a non-zero result for infinite arguments, but the GCC optimized version applies the sign of its infinite argument to its return value, whereas glibc always returns +1.
